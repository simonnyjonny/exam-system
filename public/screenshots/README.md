# Screenshots

This directory contains screenshots for the GitHub portfolio.

## Recommended Screenshot Sizes

- **Hero/Overview**: 1200x800px
- **Feature screenshots**: 800x600px
- **Mobile views**: 375x812px (iPhone SE)

## Naming Convention

Use kebab-case for file names:
- `login-page.png`
- `student-dashboard.png`
- `admin-questions.png`
